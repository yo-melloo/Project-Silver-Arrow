# 🏎️ Roadmap de Evolução - Project Silver Arrow

O projeto é dividido em 4 fases principais, partindo da simulação virtual para a telemetria real.

---

### ✅ Fase 1: Coleta e IPC (Python)
- Extração de dados via Shared Memory (mmap/struct) do Assetto Corsa.
- Interface em terminal e mascoste Argentum Buddy.

### 🚥 Fase 2: Hardware e Web (ESP32/WebSocket)
- Implementação do Argentum Bridge (WebSockets API).
- Integração de Shift-Lights físicas com ESP32.
- Sincronismo com o ecossistema OpenRGB.

### 🌐 Fase 3: Dashboard Web (Next.js)
- Interface de alta fidelidade para visualização gráfica.
- Análise de telemetria em tempo real no navegador.

### 🚗 Fase 4: Real World Telemetry (OBD-II/CAN Bus)
- Conexão com o protocolo OBD-II de veículos reais.
- Adaptação do Argentum para interfaces automotivas físicas.
